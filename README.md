# 高项学习器 · Gaoxiang Learner

> 信息系统项目管理师（软考高项）过程可视化学习器 + 论文专区
>
> 把十大知识域、五大过程组、绩效域、IT 新技术、法规标准、论文五段式模板全部做成一张「可翻可练可背」的交互式卡片墙。

一个纯前端单页应用，**打开 `index.html` 就能用**，不需要任何构建和后端。适合备考路上随时翻、随时练、随时背。

![高项学习器界面预览](docs/screenshot.png)

---

## ✨ 功能亮点

### 📐 项目管理模块
- **十大知识域** × **五大过程组**：整合、范围、进度、成本、质量、资源、沟通、风险、采购、干系人
- **7 种学习视图**自由切换：
  - 过程组总览 · 学习卡 · 关键词速查 · 场景演练 · 公式看板 · 概念对比 · 练习场
- **绩效域**（Performance Domains）单独视角，呼应新版考纲
- **盲填模式** / **关键词翻卡**：强制回忆，拒绝"看过即会"的错觉
- **学习进度标记**：已学过的过程一键勾选，复习路径一目了然

### 💡 IT 技术 & ⚖️ 法规标准模块
- 新一代信息技术（云/大数据/AI/物联网…）、信息化相关知识要点
- 法律法规、标准规范、职业道德汇总

### ✍️ 论文专区（重点）
- **考试基本规则** · **五段式结构模板** · **评分档位与易错点**
- **十大知识域写作要点**：每个域给出项目切入句、关键过程、金句收尾
- **项目素材库**：可直接套用的项目背景模板
- **金句表**：按段落/知识域分类的万能句
- **历年真题**整理
- **我的论文**：沉淀自己的项目背景、各段落字数要求、按知识域/绩效域分组的论文提纲

### 🎯 其他小心机
- **考试倒计时**（默认 2026-05-24，可在 `app.js` 改）
- **全文搜索**：过程名、关键词、公式一键定位
- **移动端适配**：地铁上也能刷

## 🚀 快速开始

### 方式一：本地直接打开
```bash
git clone https://github.com/<your-name>/gaoxiang-learner.git
cd gaoxiang-learner
open index.html        # macOS
# 或 Windows 直接双击 index.html
```

### 方式二：起个本地服务器（推荐，避免浏览器对本地文件的限制）
```bash
# Python 3
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

### 方式三：GitHub Pages 托管
1. 推送到 GitHub
2. 仓库 `Settings` → `Pages` → Source 选 `main` 分支根目录
3. 拿到一个 `https://<your-name>.github.io/gaoxiang-learner/` 的公网地址，手机也能刷

## 🗂️ 项目结构

```
gaoxiang-learner/
├── index.html                 # 主页面（Vue 3 模板）
├── app.js                     # Vue 应用逻辑（视图切换、搜索、练习场等）
├── styles.css                 # 自定义样式（Tailwind 之外的细节）
└── data/                      # 全部知识数据（纯 JS 对象，易改易扩）
    ├── knowledge.js           # 知识域总入口（汇总所有 domain）
    ├── performanceDomains.js  # 绩效域
    ├── pm_overview.js         # 项目管理基础
    │
    ├── integration.js         # 整合管理
    ├── scope.js               # 范围管理
    ├── schedule.js            # 进度管理
    ├── cost.js                # 成本管理
    ├── quality.js             # 质量管理
    ├── resource.js            # 资源管理
    ├── communications.js      # 沟通管理
    ├── risk.js                # 风险管理
    ├── procurement.js         # 采购管理
    ├── stakeholder.js         # 干系人管理
    │
    ├── it_newtech.js          # 新一代信息技术
    ├── it_informatization.js  # 信息化
    ├── law_regulations.js     # 法规与标准
    │
    ├── essay.js               # 论文专区（模板 / 评分 / 素材 / 金句 / 真题）
    └── my_essay.js            # 我的论文（个人项目背景 + 提纲沉淀）
```

## 🛠️ 技术栈

| 类别 | 选型 | 备注 |
| --- | --- | --- |
| 框架 | **Vue 3** (CDN) | 无构建，`script` 标签直接引 |
| 样式 | **Tailwind CSS** (CDN) + 自定义 CSS | `styles.css` 里补了动画、卡片阴影等 |
| 数据 | 纯 JS 对象（`window.xxxData`） | 无数据库，改 `data/*.js` 即可 |
| 部署 | 静态页面 | 本地 / GitHub Pages / Vercel / Netlify 都行 |

**零依赖、零构建、零后端** —— 这就是它能"克隆下来双击就能用"的原因。

## 📝 自定义 & 扩展

### 改考试倒计时
编辑 `app.js`：
```js
examDate: new Date("2026-05-24T09:00:00"),
```

### 加一个新的知识点 / 过程
找到对应的 `data/xxx.js`，按已有结构往数组里追加即可，页面会自动渲染。

### 沉淀自己的论文
编辑 `data/my_essay.js`：填上你的项目背景、各知识域/绩效域的写作提纲，考场之前回来抄自己就行。

## 🎨 设计小思路

- **配色**：以 `ink / ocean / lagoon / mint` 为主的冷静蓝绿，长时间看不累；`coral` 做强调色
- **信息密度**：支持「核心模式 / 完整模式」切换，冲刺期只看核心，复盘期看完整
- **强制回忆优先**：盲填、翻卡、连连看，而不是"划重点 + 通读"
